# IOPA Specification 3.0 Experimental Released

<div className="text-sm mt-2 text-center text-gray-500 dark:text-gray-400">

</div>

<Authors authors={['guybarnard']} />

IOPA 3.0 is an experimental version that starts to add some TypeScript syntax sugar.

It is deployed in production but the approach has been hardened into the emerging 4.0 core which will also
add common web standards for headers, body, blobs, etc.
